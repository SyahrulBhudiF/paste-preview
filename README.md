# Paste Preview

Share code and markdown snippets via public links. Built with TanStack Start, deployed on Cloudflare Workers.

## Stack

- TanStack Start + React 19
- Effect-TS
- Tailwind v4
- Cloudflare Workers + KV

## Development

```sh
vp install   # install deps
vp dev       # start dev server
```

## Deploy

```sh
vp run deploy
```
